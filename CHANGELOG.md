# Changelog

## [2.3.0] - 2026-05-05

### Added
- Завантаження `anchors.txt` через браузер на сторінці інструментів — більше не потрібен FTP-доступ

---

## [2.2.1] - 2026-05-05

### Fixed
- Відносні URL з Google Sheets (`/services/...`) тепер автоматично доповнюються `home_url()` перед валідацією — раніше всі такі рядки відхилялись і таблиця вважалась порожньою

---

## [2.2.0] - 2026-05-05

### Changed
- Повернуто архітектуру 2 сторінок: `relinks-options` (ACF налаштування) і `relinks-tools` (інструменти)
- Сторінка інструментів прихована з меню, доступна через кнопку "→ Синхронізація та інструменти" на сторінці налаштувань
- Sync і Import форми тепер на окремій сторінці — без конфліктів з ACF form

### Fixed
- Кнопка синхронізації більше не сабмітить ACF форму (root cause: nested forms всередині ACF options page)
- Видалені всі тимчасові костилі (GET link, transients, JS appendTo)

---

## [2.1.3] - 2026-05-05

### Fixed
- Повідомлення після синхронізації тепер зберігається через transient замість URL-параметрів — виправлено проблему з втратою notice через ACF redirect

---

## [2.1.2] - 2026-05-05

### Fixed
- Кнопка "Синхронізувати" тепер рендериться як посилання з nonce замість вкладеної форми — виправлено конфлікт з ACF form, що спричиняв збереження сторінки замість синхронізації

---

## [2.1.1] - 2026-05-05

### Fixed
- Секція статистики та імпорту тепер коректно вставляється в `.wrap` через JS замість рендеру у футері сторінки

---

## [2.1.0] - 2026-05-05

### Added
- Автооновлення плагіну з GitHub через plugin-update-checker v5.6
- Кнопка "Налаштування" на сторінці /wp-admin/plugins.php
- Кнопка "↻ Синхронізувати" безпосередньо під полем Google Sheets URL

### Changed
- Сторінку інструментів (relinks-tools) об'єднано з Options: статистика анкорів і стан бази тепер відображаються внизу сторінки Relinks Options
- Бібліотека plugin-update-checker включена в репозиторій (lib/)

### Fixed
- Посилання кнопки Settings тепер веде на коректний URL (options-general.php?page=relinks-options)

---

## [2.0.0] - початкова версія

### Added
- ACF Block "Блок перелінковки" з рендером на фронтенді
- Автогенерація посилань з рівномірним розподілом (обов'язкові URL першими)
- Синхронізація анкорів з Google Sheets (CSV)
- Імпорт з anchors.txt як ручний fallback
- Статистика використання анкорів в адмін-панелі
- Глобальні налаштування: обов'язкові URL, GSheets URL, Custom CSS, вмикач системи
- Поля на рівні поста: автогенерація, кількість посилань, ручний список
