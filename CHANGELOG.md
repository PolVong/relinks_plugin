# Changelog

## [2.1.1] - 2026-05-05

### Fixed
- Секція статистики та імпорту тепер коректно вставляється в `.wrap` через JS замість рендеру у футері сторінки

---

## [2.1.0] - 2026-05-05

### Added
- Автооновлення плагіну з GitHub через plugin-update-checker v5.6
- Кнопка "Налаштування" на сторінці /wp-admin/plugins.php
- Кнопка "↻ Синхронізувати" безпосередньо під полем Google Sheets URL

### Changed
- Сторінку інструментів (relinks-tools) об'єднано з Options: статистика анкорів і стан бази тепер відображаються внизу сторінки Relinks Options
- Бібліотека plugin-update-checker включена в репозиторій (lib/)

### Fixed
- Посилання кнопки Settings тепер веде на коректний URL (options-general.php?page=relinks-options)

---

## [2.0.0] - початкова версія

### Added
- ACF Block "Блок перелінковки" з рендером на фронтенді
- Автогенерація посилань з рівномірним розподілом (обов'язкові URL першими)
- Синхронізація анкорів з Google Sheets (CSV)
- Імпорт з anchors.txt як ручний fallback
- Статистика використання анкорів в адмін-панелі
- Глобальні налаштування: обов'язкові URL, GSheets URL, Custom CSS, вмикач системи
- Поля на рівні поста: автогенерація, кількість посилань, ручний список
